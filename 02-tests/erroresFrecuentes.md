# Errores frecuentes

---

# Redes

## Error
Confundir switch con hub

## Corrección
Hub replica tráfico a todos los puertos.
Switch aprende MACs y envía tráfico sólo al puerto correspondiente.

---

# Seguridad

## Error
SSL/TLS poco claros

## Corrección
SSL obsoleto.
TLS es el estándar moderno de cifrado seguro sobre comunicaciones.

---

# Bases de datos

## Error
INNER JOIN poco preciso

## Corrección
INNER JOIN devuelve únicamente registros coincidentes entre ambas tablas.
